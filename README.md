# Cartes d'armes (et d'armures - bientôt)
Un deck comprenant les armes et armures de base ainsi que leur principales caractéristiques pour les jeux de rôle 5E.

## Les principales informations pour vos personnages

Ces cartes comportent les caractéristiques principales des différentes armes les plus courantes : prix, dégâts, propriétés, etc. Vous trouverez également une rapide description de l'arme par Odgärd Main-de-Fer, célèbre forgeron de la capitale à l'humour parfois douteux.
